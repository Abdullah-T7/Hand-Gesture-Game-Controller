# Hand-Gesture-Game-Controller
A hand controller, is used to control games.


HAND GESTURE GAME CONTROLLER 

A virtual XBOX 360 controller that is used to controll pc games, there is not need to connect a real controller for pc.

PROJECT OVERVIEW:
> Play games using our hand gestures
> Convert hand movement into different joystick and keyboard movement
> Control game like Trackmania
> Using of different libraries of python

TECHNOLOGIES USED:
> Python 3.9
> Vgamepad (To make virtual XBOX Controller)
> Mediapipe (Ai powered library for detection of hands)
> pynput (For keyboard control)
> Math (For Joystick normalizaiton)
 
HOW IT WORKS:
> Both hands are introduced
> The right hand is used to control joystick movement
> The Left hand is used to control the chracter (If the hand is open the chracter move forward and backward, if the hand is close)
> 'Q' is used to quit the window
> Work better in good light conditions

FEATURES:
> Real-Time hand tracking
> Uses both hands for better control of games
> Game tested (AS Trackmania)

SUPPORTED GAMES:
Games that support Xbox / XInput controllers, for example:
> Trackmania
> Need more adjustment
