# DISCRIPTION
A hand controller, is used to control games.

PROJECT OVERVIEW:
> Track your hands in real time using a webcam
> Convert hand movements into joystick input
> Control games like Trackmania
> Use AI-powered hand landmarks instead of buttons

TECHNOLOGIES USED:
> Python 3.9
> Mediapipe (Used for tracking hands)
> pynput (For controlling games on keyboard)
> Vgamepad (A python library for virtual XBOX controller)
> Math (For game normalization)

HOW IT WORKS:
> Both hands are used to control the game chracter
> The right hand is used to control the joystick 
> The left hand is used to control the chracter (open hand for moving forward and close for backward)
> Press 'q' to quit the game

FEATURES:
> Real-time hand tracking
> Uses both hands for better control
> Supports real games (Trackmania tested)

SUPPORTED GAMES:
Games that support Xbox / XInput controllers, for example:
> Track Mania
> Other games with small improvement (depending on the controls of game).
